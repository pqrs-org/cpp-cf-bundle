# cpp-cf-bundle

Utilities of `CFBundle`.

## Requirements

cpp-cf-bundle depends the following classes.

- [pqrs::cf::url](https://github.com/pqrs-org/cpp-cf-url)

## Install

Copy `include/pqrs` and `vendor/vendor/include` directories into your include directory.
